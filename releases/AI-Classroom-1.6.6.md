# AI Classroom 1.6.6

## Updates

- Moved main classroom memory and chapter organization to a delayed background task.
- Removed per-message memory summarization from the foreground chat path.
- Memory now waits for a batch of messages before rebuilding chapters and summaries.
- Chat replies are saved and shown immediately; background memory indexing is persisted afterward.
