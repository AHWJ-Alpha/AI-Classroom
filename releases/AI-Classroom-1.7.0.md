# AI Classroom 1.7.0

## 版本说明

- 新增更新检测模块：应用启动后会检查 GitHub Release 是否存在新版本。
- 检测到新版本时会弹窗提示更新，同一天最多提示一次。
- 如果 GitHub 或相关网络连接失败，应用会静默跳过，不弹窗打扰学习。
- 优化考试模式：横屏、竖屏切换时不会退出当前考试。
- 竖屏考试只显示题目、答案、提交等核心内容。
- 横屏考试才提供草稿纸和过去总览，并且两个面板默认关闭，只能通过按钮打开。
- 草稿纸支持“打字”和“涂绘”两种模式，可随时切换，涂绘模式支持清空。
- 优化考试界面的按钮、间距和侧栏布局，让答题界面更清爽。

## 近期体验优化

- 发送按钮移动到输入框右下角。
- “到底部 / 到开头”按钮上移，减少与输入区冲突。
- 记忆和章节整理改为后台批量执行，不再影响前台对话速度。
