# AI Classroom 1.6.5

## Updates

- Improved the contrast between user message bubbles and the main classroom background.
- Removed the dark overlay block beside the secondary classroom menu.
- Added a stricter `[EXAM]...[/EXAM]` exam protocol so the AI can trigger the immersive exam module in real conversations.
- Added fallback exam detection: when the user clearly asks for a quiz, test, mock exam, or practice questions, the app can still open the exam module even if the model output is not perfectly formatted.
- Hidden machine-readable exam blocks from normal classroom chat display.
