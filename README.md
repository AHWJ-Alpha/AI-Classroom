# AI Classroom

AI Classroom 是一款面向长期学习场景的 Android AI 课堂 Demo。它围绕“主课堂 + 分支课堂 + 长期记忆 + 知识库”的学习模式，让用户可以在 AI 导师陪伴下持续学习一门课程，并在不打断主线的情况下展开支线探索。

当前最新版本：`2.1.1`

## 核心功能

### 主课堂

- 用户可自定义学习内容。
- AI 会围绕当前课堂持续教学。
- 支持 Markdown 和数学公式文本显示。
- 支持真实流式输出，AI 回复会边生成边显示。
- 支持在主课堂上传照片，由多模态模型分析解答。
- 课堂、对话、配置、记忆和知识库均保存在本地。

### 分支课堂

- 可从主课堂任意一段对话开启分支。
- 分支是与主课堂平行的独立长对话。
- 开分支时会冻结当时的主课堂上下文。
- 分支后续内容只保存在分支内，不会复制、打包或改写主课堂。

### 长期记忆

- 后台按批次整理对话，不阻塞前台聊天。
- 自动生成章节标题和一句话摘要。
- 记忆页支持章节索引。
- 全览模式会生成实时向量思维导图，章节按摘要关键词相似度自动连接。

### 知识库

- 支持直接读取 `.md` 和 `.txt` 文件。
- 文件内容会作为课堂知识库上下文参与教学。

### 模型与 API 配置

- 模型模块、多模态模块、TTS 模块分别拥有自己的 Base URL、API Key 和模型配置。
- 模型模块支持多个模型按优先级自动降级。
- 支持深度思考模式，可单独配置深度思考模型。
- 多模态模块用于课堂图片分析。
- TTS 模块可保存语音模型和音色配置。

### 考试模式

- 当 AI 明确输出结构化考试内容时，应用会进入沉浸式考试界面。
- 题目支持“前提 + 问题 + 答案框”的结构。
- 提交后会退出考试界面，并将答案打包给 AI 批改。
- 标记“不会”的题目会生成对应讲解分支。

### 主题和界面

- 默认二次元青蓝配色：`#39C5BB` 和 `#00AEEF`。
- 支持跟随系统、黑白、单色和二次元主题。
- 支持自定义主色和辅色。
- 导航栏、按钮、输入框、筛选项和卡片会统一跟随当前皮肤。
- 设置页最多展开一个模块，保存后自动收起当前模块。

## 本地编译

使用 Android Studio 打开项目，或在项目根目录运行：

```powershell
.\gradlew.bat assembleDebug --no-daemon
```

Debug APK 输出位置：

```text
app/build/outputs/apk/debug/app-debug.apk
```

项目内归档 APK 位于：

```text
releases/
```
